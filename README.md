# create-bootstrap-app
Create bootstrap is CLI tool to create a boilerplate template for bootstrap 4 app.
```bash
    -index.html
    -script.js
    -style.css
```

## Install
```bash
npm i -g @suboorkhan/create-bootstrap-app
```

## Usage
```bash
create-bootstrap-app
```
